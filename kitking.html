Kitking is a website for selling football kit attire. I was tasked with redesigning the website with a focus on usability.

View the full presentation here.

Indentifying the issues.

I began by identifying the key audience of the attire and then performed an expert evaluation applying Smashing UX’s heuristics. Some general issues were summarised:

  • Visual Clutter
  • Visual Inconsistency
  • Layout Inconsistency
  • Poor Accessibility
  • Unattractive Design

    • I then got the website in front of 5 target audience users of the website (young adolescents with budgetary constraints) and performed guerrilla testing – giving the users key tasks to perform on the interface (see video). They identified these core issues:

  • Visual Clutter
  • Search not Working
  • Difficulty Finding Things
  • No Price Filters
  • Poor Aesthetics

    • Solving The Issues.

    I attempted to address some of the issues using A/B testing with a focus on removing visual clutter and making the website more visually appealing. One such example:

    This used 5 second test and I performed these tests across multiple page elements using 40 participants at a time. The proposed designs were received overwhelmingly positively. From here, I applied this knowledge and data gathered previously from users and expert evaluation to redesign the website with an implicit focus on a simplified, aesthetically pleasing design, using the redesigned elements identified in A/B Testing. This resulted in this design (designed principles used explored in the presentation):

    As of now, the client hasn't made any radical changes to the design, however some of the clear accessibility issues outlined in the presentation have been broadly addressed, including a new navigation menu, redesigned 'kit search' function and filters for listing through clothing.
